# ave-tesk-task
Тестовое задание для Ave Technologies

# Запуск приложения
1. **Клонировать репозиторий:**

   ```bash
   git clone git@github.com:sva24/ave-tesk-task.git
   ```
2. **Перейти в директорию проекта:**

      ```bash
   cd ave-tesk-task
      ```
3. **Запустить докер контейнер:**

      ```bash
   docker compose up --build -d
   ```
